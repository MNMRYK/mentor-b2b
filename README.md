# mentor-b2b
Concepto: SPA corporativa (Navy/Dorado) enfocada en conversión B2B de alto valor. Incluye 4 micro-interacciones JS nativas: Calculadora de ROI en tiempo real, filtro dinámico de productos sin recarga, formulario de cualificación condicional (tipo Typeform) y barra de progreso animada en el dashboard del cliente.
